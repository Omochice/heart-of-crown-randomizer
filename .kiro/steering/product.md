# Product Overview

A web-based randomizer tool for the card game "Heart of Crown". Helps players create randomized game setups by generating card selections from various expansions and sets.

## Core Capabilities

- Card data management across game expansions (Basic, Far Eastern Border)
- Card categorization by type (common, rare, princess)
- Randomized card set generation for game setup
- Web-based interface for easy access

## Target Use Cases

- Players setting up new Heart of Crown games who want variety through randomization
- Game groups looking for balanced, unpredictable card distributions
- Solo players testing different card combinations

## Value Proposition

Streamlines game setup by automating the card selection process while maintaining game balance and fairness through proper categorization and randomization rules.
